[국문 초록]

컴퓨터 비전 모델 개발에서 데이터 수집 및 라벨링은 전체 파이프라인의 상당 부분을 차지하는 주요 병목 공정이다. 기존 자동 라벨링 도구들은 고정된 파이프라인과 단일 모델 의존성으로 인해 이종 데이터 형식 대응이 어렵고, 생성된 라벨의 품질 검증 및 데이터 불균형 문제 해결에 있어 여전히 사용자의 수동 개입을 요구하는 한계가 있다. 특히, 단일 대형 언어 모델(LLM)에 의존한 품질 검증(Self-Reflection)은 모델 자체의 편향으로 인해 자기 오류를 식별하지 못하거나 오히려 증폭시키는 한계가 존재한다.

본 논문은 이러한 한계를 극복하기 위해 서로 다른 파라미터 규모(Capacity)를 가진 이종 대형 멀티모달 모델(Heterogeneous LMM)을 계층적으로 연결한 'Agentic Workflow 기반 자동 라벨링 및 검증 시스템'의 아키텍처를 제안한다. 제안하는 시스템은 자연어 명령을 해석하여 데이터를 타겟 규격으로 자율 변환하는 라벨링 에이전트(Labeling Agent), 다중 반복 추론 간의 기하학적 일관성(IoU)을 통해 불확실성(Uncertainty)을 측정하고 임계치 미달 샘플만 고성능 LMM에 에스컬레이션(Escalation)하는 계층적 검증 에이전트(Hierarchical Verification Agent), 그리고 클래스 분포를 분석하여 증강 전략을 제안하는 데이터셋 인사이트 에이전트(Dataset Insight Agent)의 3-Agent 구조로 설계되었다.

제안하는 계층적 캐스케이드(Cascade) 구조는 Ground Truth가 없는 상황에서도 모델의 자가 일관성(Self-consistency)을 바탕으로 라벨의 신뢰도를 평가하도록 고안되었다. 이를 통해 불확실한 데이터에만 고성능 연산 자원을 집중적으로 할당함으로써, 전체적인 API 호출 비용을 통제하는 동시에 라벨 데이터의 품질을 최적화할 수 있을 것으로 기대된다. 본 연구는 이종 모델 간의 유기적인 협업을 통한 에이전트 기반 데이터 큐레이션이라는 새로운 패러다임을 설계했다는 데 의의가 있으며, 비전 AI 데이터 준비 단계의 확장성과 신뢰성을 크게 향상시킬 수 있는 이론적 토대를 제공한다.
