[2] 국문 초록 (제출용)
--------------------------------------------------------------------------------
컴퓨터 비전 모델 개발에서 데이터 수집 및 라벨링은 전체 개발 시간의 상당
부분을 차지하는 병목 공정이다. 기존 자동 라벨링 도구들은 고정된 파이프라인과
단일 모델 의존으로 인해 이종 데이터 형식 대응, 라벨 품질 검증, 데이터
불균형 대응 측면에서 사용자의 수동 개입을 요구하는 한계가 있다.

본 논문은 이러한 한계를 극복하기 위해 LLM 기반 Agentic Workflow로 동작하는
통합 자동 라벨링 및 검증 시스템을 제안한다. 제안 시스템은 사용자의 자연어
명령을 해석하여 이종 포맷 데이터의 구조를 자율적으로 추론하고 타겟 모델
규격으로 변환하는 라벨링 에이전트, 경량 VLM으로 1차 라벨을 생성한 뒤
불확실 영역에 대해서만 고성능 LMM(GPT-4o, Claude 3.5 Sonnet 등)에 재검증을
요청하는 계층적 자가 검증(Hierarchical Self-Reflection) 에이전트, 클래스
분포를 분석하여 불균형을 탐지하고 증강 또는 추가 데이터 수집 전략을
제안하는 데이터셋 인사이트 에이전트로 구성된다. 특히 계층적 검증 구조는
단일 모델 self-reflection의 한계를 극복하면서 전체 API 비용을 제어한다.

제안 시스템을 [데이터셋명]에 적용한 결과, 수동 라벨링 대비 작업 시간을
약 [X]% 단축하고 라벨 오류율을 [Y]% 감소시켰다. 본 연구는 비전 AI 개발의
데이터 준비 병목을 구조적으로 해소하고, 에이전트 기반 데이터 큐레이션
패러다임을 제시한다는 점에서 의의가 있다.


[3] 핵심 방어 논리 (리뷰어 Q&A 대비)
--------------------------------------------------------------------------------
Q: 본 논문의 novelty는 무엇인가?
A: "단일 LLM의 self-reflection이 자기 오류를 잘 못 잡는다는 한계가
   알려져 있는데, 본 연구는 서로 다른 capacity의 LMM을 계층적으로 연결하여
   모델 간 이질성(heterogeneity)을 활용했고, 모든 샘플이 아닌 불확실한
   샘플만 에스컬레이션하여 비용을 제어했다는 점이 차별점입니다."

Q: 기존 자동 라벨링 도구(Autodistill, Roboflow 등)와 무엇이 다른가?
A: "기존 도구들은 단일 모델을 사용하거나 고정된 파이프라인이지만, 본 시스템은
   자연어 명령 기반의 agentic workflow로 동작하며, 불확실성 기반 계층적
   검증을 통해 품질과 비용을 동시에 제어한다는 점이 다릅니다."

Q: 불확실성은 어떻게 측정하는가?
A: "1차 모델의 confidence score와 bounding box의 기하학적 일관성
   (예: 동일 객체에 대한 다중 detection의 IoU)을 결합하여 산출합니다.
   임계값 이하의 샘플만 고성능 LMM에 재질의합니다."

불확실한 샘플 : 요소 1: Confidence Score (모델 자체의 확신도)
, 요소 2: 기하학적 일관성 (Geometric Consistency) : (Multiple Inference (같은 이미지 여러 번)

동일 이미지를 VLM에 3번 입력 (temperature를 약간 올려서 다양성 확보)
각각 bounding box를 받음
3개의 box가 서로 얼마나 겹치는지 IoU로 측정
IoU가 높으면(예: 0.9 이상) → 일관성 있음 → 신뢰
IoU가 낮으면(예: 0.5 미만) → 모델이 헷갈려함 → 불확실) 
이정도 가이드라도 주자.

< 현재 초록 문제점 >
1. Novelty 불명확 : 기존 자동 라벨링 도구(Autodistill, Roboflow 등)와의 차별점이 없음.
2. 문제 정의 분산 : 포맷 변환(사소함)과 품질 검증(중요함)이 섞여 핵심이 흐려짐.
3. IoU 검증 기준 모호 : "무엇과 무엇의 IoU"인지 불명확. Ground Truth 없는 상황에서 순환 논리 우려.
4. Self-Reflection 근거 부족 : 단일 모델 self-reflection은 오히려 오류를 증폭시킨다는 연구 결과 존재.
5. 실험 결과 없음. : 정량 수치가 하나도 없음. "proposal" 수준에 머무름.

< 추천 초록 핵심 요지 >
1. 핵심 기여 (Novelty)
서로 다른 capacity의 LMM을 계층적으로 연결한 Heterogeneous Model Cascade 구조. 
단일 모델 self-reflection의 한계를 극복하고, 불확실 샘플만 고성능 모델에 에스컬레이션하여 품질과 비용을 동시에 제어.

2. 문제 정의 (기존 도구의 3가지 한계)
(1) 단일 모델 의존 → 품질 검증 한계
(2) 고정 파이프라인 → 이종 포맷 대응 한계
(3) 수동적 데이터 관리 → 불균형 대응 부재

3. 3-Agent 구조 (한계 1:1 대응)
(1) Labeling Agent → 이종 포맷 자동 변환
(2) Hierarchical Verification Agent → 계층적 검증 (핵심)
(3) Dataset Insight Agent → 클래스 분포 분석 및 증강 전략 제안

4. 불확실성 측정 기준
반복 예측 간 IoU 일관성 + confidence score를 결합한 Uncertainty Score.
Ground Truth 불필요 (self-consistency 기반).

5. 정량 결과 필수 확보
시간 단축률, 라벨 정확도, 계층적 검증 전후 오류율 비교(ablation).
COCO/VOC subset으로 하루 실험 가능.
